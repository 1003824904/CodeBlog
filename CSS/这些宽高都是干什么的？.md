# 这些宽高都是干什么的？

在前端开发中，一系列的宽度困扰着我们。很多都容易弄混淆，下面我们一次把他们区分开来。

## clientWidth和clientHeight

`Element.clientWidth/clientHeight`

> Element.clientWidth/clientHeight 属性表示元素的内部宽度/高度，以像素计。该属性包括内边距，但不包括垂直滚动条（如果有）、边框和外边距。

**只读属性**

-  `element.clientHeight`
